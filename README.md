# Apollo Computing History

*Date: 2026-03-19*

## Apollo Guidance Computer (AGC)

The Apollo Guidance Computer (AGC) was a digital computer developed for NASA's Apollo program, installed on the command module and lunar module. Introduced in August 1966, it featured:
- 2.048 MHz processor using discrete silicon integrated‑circuit chips (RTL).
- 15‑bit word length with 1‑bit parity.
- 2048 words of magnetic‑core RAM and 36,864 words of core rope ROM.
- Power consumption of about 55 W and weight of ~70 lb (32 kg).
- Operated via the DSKY (Display and Keyboard) interface.
- Software written in AGC Assembly Language.

## Recent Developments

*The Apollo code restoration project* aims to preserve and modernize the original AGC source code. Recent efforts include:
- 2024 release of a fully annotated source repository on GitHub (github.com/astronautics/AGC).
- 2025 collaboration between NASA and the Smithsonian to create an interactive web‑based emulator.
- Ongoing community-driven documentation and translation of AGC assembly to modern C for educational use.
